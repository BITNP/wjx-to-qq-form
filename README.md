# 问卷星 → 腾讯收集表 → 腾讯文档

从问卷星导出表格，将新数据抄到腾讯收集表，以同步到腾讯文档中。

## 使用方法

首先创建`config.toml`，记录各种地址。

```toml
# 问卷星管理后台页面地址中“activity=”后的一串数字
wjx = "299792458"
# 腾讯收集表填写页面中“/form/page/”后的一串字母数字
qq-form = "MetersPer5ec0nd"
```

然后运行`pnpm run main`。（初次使用需要登录）
